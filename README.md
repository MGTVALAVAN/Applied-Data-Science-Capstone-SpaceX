# SpaceX Falcon 9 First Stage Landing Prediction

## Project Overview
This project is part of the IBM Applied Data Science Capstone. It predicts the success of Falcon 9 first-stage landings using historical launch data.

## Data Sources
- SpaceX REST API
- Wikipedia Falcon 9 launch records

## Project Components
- Data Collection and Wrangling
- Exploratory Data Analysis (EDA)
- SQL Analysis
- Interactive Visualization (Folium)
- Dashboard using Plotly Dash
- Machine Learning Classification

## Models Used
- Logistic Regression
- Support Vector Machine (RBF Kernel)
- Decision Tree
- K-Nearest Neighbors
